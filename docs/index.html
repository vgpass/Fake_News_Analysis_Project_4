INTRODUCTION

For Project 4 on machine learning, we chose to train a model to analyze whether a news article was real or fake. Our original dataset (link provided on resources page) had 6,335 articles with columns for the title, article text, and label. The articles were labeled as ‘real’ or ‘fake.’ Cleaning of text data at its most basic consists of normalizing the text and then removing ‘stopwords.’ Normalizing is removing capitalization, Unicode characters, and punctuation. ‘Stopwords’ are words that don’t have relevance in text interpretation, such as ‘a,’ ‘an,’ and ‘the.’ We used the Natural Language Toolkit (NLTK), a python text cleaning library, to clean the text.

Three models were applied to the cleaned text data: Logistic Regression, Linear Support Vector (LSV), and Keras-Bidirectional Encoder Representations from Transformers (BERT).

Once the code for cleaning was complete and the models run on the smaller data set, we applied our code to a more robust dataset comprised of 51,233 articles, rough split in half between fake and real articles.

Basic information on the dataset is below with links to our results in the sidebar. Our repository with complete code, data files, and databases can be found at GitHub Fake News Analysis Project 4
