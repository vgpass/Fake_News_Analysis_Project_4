Model #1 - Logistic Regression

The module sklearn.linear_model is a valuable tool for machine learning practitioners, providing a range of linear models that can be applied to regression, classification, and other tasks. These models include:

  • LinearRegression: A basic linear regression model that predicts continuous target variables based on input features.

  • LogisticRegression: A model that is particularly useful for binary classification tasks, where the aim is to predict one of two classes.

  • Ridge: A linear regression model that employs L2 regularization to avoid overfitting and improve generalization.

  • Lasso: A linear regression model that utilizes L1 regularization to encourage sparsity in the model's coefficients, leading to more interpretable models.

  • ElasticNet: A linear regression model that blends L1 and L2 regularization techniques to provide a balanced approach between Ridge and Lasso regularization.

  • SGDClassifier and SGDRegressor: Models that use Stochastic Gradient Descent (SGD) for optimization, facilitating efficient training on large datasets.

    • For our first machine learning model, we decided to use logistic regression, a suitable fit for predicting whether a news article is real or fake.
